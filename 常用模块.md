### random

| random              | 取值范围                  | 数据类型 |
| ------------------- | ------------------------- | -------- |
| random.randint(a,b) | [a,b]                     | 整数     |
| random.random()     | 0 <= n < 1.0              | 浮点数   |
| random.randrange()  | [start, stop[, step])     | 数       |
| random.choice()     | list，tuple，字符串等序列 | 字符     |
|                     |                           |          |
